Change app
=========================


## Prerequisites

1. Node 14.0.0+
2. Python 3.6 or above

## Getting Started

1. Clone the repository
2. Change in to the directory that was cloned

## Start api
1. `cd api`
2. `python -m pip install --user -r requirements.txt`
3. `python calc_change.py`
4. Api available on port 5000

## Start web app
1. `cd ..`
2. `npm install`
3. `npm build`
3. `npm start`
4. Web app available at http://localhost:3000/

## Unit tests

Unit tests are written in [Jest](https://jestjs.io). Run them with `npm test`.
